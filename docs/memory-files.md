# Memory Files

`.project-memory/` contains editable project memory.

Files:

- `summary.md`: what the project does
- `commands.md`: run/test/build/lint commands
- `architecture.md`: folders, modules, data flow
- `style.md`: coding style and assistant rules
- `known_errors.md`: recurring errors and fixes
- `last_session.md`: latest working state
- `decisions.md`: design decisions
- `todos.md`: task state

`AI_MEMORY.md` is generated from these files and should be attached to your coding assistant.
